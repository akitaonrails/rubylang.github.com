    Downloads

  • Usando RVM

    Muitos rubistas preferem instalar Ruby via RVM, o Gerenciador de Versões de Ruby. Ela torna o processo de instalar Ruby incrivelmente fácil e ainda permite instalar mais de uma implementação de Ruby como JRuby, Rubinius, MacRuby, MagLev, e IronRuby.

    RVM está disponível apenas para Mac OS X, Linux ou qualquer sistema operacional estilo UNIX. Usuários de Windows devem checar o RubyInstaller, descrito na seção de Windows abaixo.

  • Mac OS X

    OS X já vem com 1.8.7, junto com muitas RubyGems. Como o 1.8.7 é uma versão antiga de Ruby, muitas pessoas instalam versões mais novas usando RVM ou Homebrew. Homebrew deixa os usuários de OS X facilmente baixar e instalar pacotes Unix a partir do código fonte.

    Homebrew somente oferece as últimas versões de Ruby, então para instalar 1.8.7 ou mais antigo você deve usar RVM.

  • Windows

    Se estiver no Windows, existe um grande projeto para ajudá-lo a instalar Ruby chamado RubyInstaller. Ele lhe dá tudo que você precisa para configurar um ambiente completo de desenvolvimento para Ruby no Windows.

    Para usar o RubyInstaller, baixe dos links abaixo e dê duplo clique no instalador.
