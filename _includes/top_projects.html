Alguns Grandes Projetos de Ruby

  • Rails Rails é um framework de aplicações web que inclui tudo que se precisa para criar aplicações web com banco de dados de acordo com o padrão Model-View-Controller (MVC).
  • Homebrew Homebrew é a maneira mais fácil e flexível de instalar ferramentas UNIX que a Apple não incluiu no OS X.
  • Jekyll Jekyll é um gerador de site estático com suporte a blog.
  • Diaspora Uma rede social de código aberto preocupado com privacidade, controlado pessoalmente, que faz de tudo.
  • Travis CI Travis é um sistema aberto de construção distribuída para a comunidade Ruby.
